DailyTasks is a simple todo list web application built as part of a programming assignment.  
It lets you create, view, and manage your daily tasks in a clean, minimal interface.

---

## Features

- Add new tasks
- Mark tasks as done (or keep them as pending)
- Delete tasks you no longer need
- Tasks stored in a local database (`todo.db`)
- Simple, responsive UI with custom CSS

---

## Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS
- **Database:** SQLite (`instance/todo.db`)
